# 좌표 구하는 방법

1. [구글지도](https://maps.google.com)에 접속하고, 좌표를 구하고 싶은 지역으로 이동한다.
2. 구하고 싶은 좌표가 있는 위치에서 우클릭을 하면 좌표값이 나타난다.
3. 해당 좌표값을 좌클릭하면 좌표가 복사된다.

# 좌표값을 프로그램에 붙여넣기

1. 프로그램으로 이동한다.
2. **Ctrl+v 또는 Ctrl+Shift+v**를 누르거나 **마우스 우클릭**을 한다.
